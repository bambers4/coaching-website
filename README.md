\# Coaching Website



A beginner-friendly static website for a personal running coaching business.



\## Description



This website introduces the coach, explains the coaching philosophy, describes available coaching services, and gives visitors a way to get in contact.



The site is built with simple HTML and CSS so it is easy to understand, edit, and expand over time.



\## Pages



\- `index.html` — Home page with the main introduction and call to action

\- `about.html` — About page with personal background, running experience, and coaching philosophy

\- `coaching.html` — Coaching services page

\- `contact.html` — Contact page with contact information and a visual contact form

\- `styles.css` — Main stylesheet for layout, colors, spacing, and responsive design



\## How to View the Website Locally



Open `index.html` in a web browser.



You can do this by double-clicking the file in your project folder.



\## How to Customize



Replace placeholder text such as:



\- `\[Your Name]`

\- `\[Your Email]`

\- `\[Your Running Background]`

\- `\[Your Coaching Philosophy]`

\- `\[Your Location]`



You can also update colors, spacing, and layout in `styles.css`.
